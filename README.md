# AI 生活教练网站

这是一个基于火山方舟 DeepSeek R1 API 开发的个人生活教练网站。通过与 AI 进行对话，获取建议和指导，帮助个人成长。

## 项目结构

- `public/index.html`: 主页面，包含聊天界面
- `public/styles.css`: 页面样式文件
- `server/server.js`: Node.js 后端服务器，处理 API 请求

## 功能特点

- 实时对话界面
- 流式响应输出
- 响应式设计，适配各种设备
- 简洁直观的用户界面

## 技术栈

- 前端：HTML5, CSS3
- 后端：Node.js
- API：火山方舟 DeepSeek R1
